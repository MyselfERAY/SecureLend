    "message": "...",

  }

  "status": "error",
  "message": "Beklenmeyen

🔐 Kimlik Dogrulama Akisi

1. POST /api/auth/login      { tckn, phone }       -> OTP gonderilir
2. POST /api/auth/verify-otp  { phone, code }       -> { accessToken, refreshToken }
3. Her istekte Header:  Authorization: Bearer <accessToken>
4. POST /api/auth/refresh      { refreshToken }     -> Yeni token cifti

Kullanici Islemleri Kilavuzu

Adim adim kullanim rehberi

🔐 Kayit & Giris

  1. Kayit

    /auth/register - TCKN, telefon, ad-soyad, dogum tarihi, rol (Kiraci/Ev Sahibi)

  2. Giris

    /auth/login - TCKN ve telefon numarasi ile giris

  3. OTP Dogrulama

    /auth/verify-otp - SMS kodu girin (dev: 111111)

  4. Dashboard

    Otomatik yonlendirme - rol bazli menu goruntulenir

👤 Kiraci - KMH Basvurusu

  1. Banka Sayfasi

    Dashboard > Banka & KMH

  2. Formu Doldur

    Istihdam durumu, aylik gelir, tahmini kira

  3. Onay Bekle

    Kriter: Gelir ≥ Kira × 2

  4. Onboarding

    Dijital kimlik dogrulama

  5. KMH Hesap

    Limit: Gelir × 3 (max 500K TL)

🏠 Ev Sahibi - Sozlesme

  1. Mulk Ekle

    Dashboard > Mulklerim > Yeni Mulk

  2. Sozlesme Olustur

    Mulk sec, kiraci TCKN, kira, IBAN

  3. Imzala (1. imza)

    Sozlesme detay sayfasi > Imzala

  4. Kiraci Imzasini Bekle

    Kiraci KMH secer ve imzalar

  5. Aktif Sozlesme

    Otomatik odeme emri olusur

✅
Imza Sureci: Ev Sahibi 1. imza → Kiraci KMH secer + 2. imza → Sozlesme ACTIVE → PaymentOrder otomatik olusur

Sik Karsilasilan Sorunlar

Hata cozumleri ve yararli ipuclari

🌐

"Failed to fetch" Hatasi

Sorun: Frontend'den API'ye istek atildiginda "Failed to fetch" hatasi.

Cozum: CORS ayarlarini kontrol edin. apps/api/src/main.ts dosyasinda frontend URL'inin origin listesinde oldugunundan emin olun.

origin: ['http://localhost:3000', 'http://localhost:3001', process.env.WEB_URL]
⏱️

Rate Limit (429) Hatasi

Sorun: Test sirasinda 429 Too Many Requests hatasi.

Cozum: app.module.ts dosyasinda ThrottlerGuard'i yorum satirina alin.

// { provide: APP_GUARD, useClass: ThrottlerGuard }
🗃️

Prisma Client Uyumsuzlugu

Sorun: Schema degisikligi sonrasi "Unknown field" hatasi.

Cozum:

cd apps/api && npx prisma generate && npx prisma migrate dev
🔒

JWT Token Suresi Dolmus (401)

Sorun: 401 Unauthorized hatasi.

Cozum: Access token 15 dakikada dolar. Refresh endpoint ile yenileyin.

POST /api/auth/refresh  { "refreshToken": "<token>" }
💰

KMH Basvurusu Reddedildi

Sorun: KMH basvurusu REJECTED olarak donuyor.

Cozum: Kriter: monthlyIncome ≥ estimatedRent × 2. Gelir tutarini en az tahmini kiranin 2 kati girin.

✏️

Kiraci Sozlesme Imzalayamiyor

Sorun: "Aktif KMH hesabiniz bulunmuyor" hatasi.

Cozum: Sirasiyla: (1) KMH basvurusu onaylanmis olmali, (2) Dijital onboarding tamamlanmis olmali, (3) KMH hesap ACTIVE olmali, (4) KMH limiti ≥ aylik kira olmali.

🔌

Port Cakismasi (EADDRINUSE)

Cozum:

lsof -i :4000    # Backend portu
lsof -i :3000    # Frontend portu
kill -9 <PID>
